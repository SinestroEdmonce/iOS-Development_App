### 应用介绍

*Kaoyaners* 是一款专为考研学子设计的集学习、资源分享、社交等于一体的App。这款App的设计目的是为了使得广大的考研人能够互相分享资源，互相分享经验，互帮互助共同奋斗在考研路上，这款App崇尚“人人为我，我为人人”的氛围，分享与自用是这款App的主题。

        虽然App设计尚未完善（譬如社交部分等）但是基本的主体框架和功能都以基本完成。
